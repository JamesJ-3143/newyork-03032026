How to Push to GitHub Step-by-Step Breakdown

  1. Initialize Local Repository (git init): In the VS Code terminal, initialize a new Git repository in your project folder
  2. Stage Changes (git add .): Stage the files to be committed, preparing them for the snapshot.
  3. Commit Changes (git commit -m "..."): Commit the staged files with a message describing the changes, which saves the snapshot locally.

Create Remote Repo & Push (git push)

  • Create a new repository on GitHub.
  • Link local to remote: git remote add origin 'URL'.
  • Push files: git push -u origin main.

Alternatively, you can use the Publish to GitHub button directly within the VS Code Source Control panel to automate the remote repository creation and initial push
